# 🤖 InfraWarriors – PeraBots 25 Simulation Robot

An autonomous simulation-based robot developed for the **PeraBots 25** competition organized by the Engineering Faculty of the University of Peradeniya.

This project was created by undergraduate engineering students from the University of Moratuwa with the goal of designing an intelligent robot capable of optimized navigation, dynamic obstacle avoidance, and efficient lap completion in a simulated environment.

---

# 🚀 Project Overview

The robot was designed to autonomously navigate through a dynamic environment while minimizing lap completion time and avoiding unnecessary path repetitions.

Our simulation system integrates:

* Real-time obstacle detection
* Dynamic path planning
* Sensor-based autonomous decision making
* Optimized lap execution
* Learning-based path efficiency improvements

The project successfully passed the simulation elimination round and was selected as a finalist project for **PeraBots 25**.

---

# ⚙️ Key Features

## 🔹 Dynamic Obstacle Avoidance

Implemented a LIDAR-based obstacle detection and avoidance system capable of reacting to environmental changes in real time.

## 🔹 Path Optimization

The robot continuously optimizes its movement path to reduce unnecessary travel distance and improve lap efficiency.

## 🔹 Autonomous Decision Making

Sensor feedback is processed in real time to allow adaptive navigation and intelligent movement control.

## 🔹 Lap Time Optimization

The system remembers previously traveled routes and minimizes redundant movements during subsequent laps.
